## Image Guidelines

### Format
- Use **JPEG** for photos and screenshots
- Use **PNG** for logos and icons with transparency
- Use **WebP** for better compression (optional)

### Size
- Keep file sizes under 500KB for faster loading
- Optimize images using tools like:
  - TinyPNG (https://tinypng.com/)
  - Squoosh (https://squoosh.app/)
  - ImageOptim (Mac)

### Dimensions
- Profile: 600x800px or similar portrait ratio
- Projects/Blog: 800x600px or 16:9 aspect ratio
- Favicon: 32x32px or 64x64px

### Quality
- Use high-quality, professional images
- Ensure good lighting and clarity
- Maintain consistent styling across all images
